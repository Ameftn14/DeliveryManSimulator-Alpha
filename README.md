### 游戏介绍

- 你扮演快递员，选择性地接订单，到餐馆取货，然后送货给客户
- 你有时间限制和背包存储限制。尽你最大努力赚更多的钱
- 有两种技能：
  - 左Ctrl键：子弹时间
  - 左Shift键：氮气加速
